## API 稳定性承诺

- **稳定端点**: GET /api/config, POST /api/config
  - 承诺在主要版本内保持向后兼容
  - 废弃时会提前 2 个次要版本通知
  
- **实验性端点**: GET /api/watch
  - 可能会在不升级主要版本时变更
  - 使用时请注意文档中的实验性标记

## 文档维护

API 文档必须与代码同步更新：
1. 新增端点时更新本文档索引
2. 修改参数时更新详细规范文档
3. 删除端点时标记为废弃，并说明替代方案
