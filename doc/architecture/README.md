# 架构设计文档

**文档类型**: 架构设计  
**适用范围**: 全版本  
**最后更新**: 2026-03-21  
**维护者**: OminiConfig Team

## 文档用途

本文档目录包含 OminiConfig 项目的所有架构设计文档，包括：
- 系统整体架构
- 模块设计说明
- 技术选型决策
- 版本演进历史

## 文档列表

### v1_overview.md

- **读者**: 想了解项目起源的开发者

## 架构演进路线图

```
v1.0 (初代)
├── 基础 JSON 读写
├── SHA256 乐观锁
└── 简单 HTTP API

v2.0 (企业级)
├── 安全沙箱（路径校验）
├── 多格式适配器（JSON/YAML/...）
├── 跨平台原子写入
└── SSE 实时推送

v3.0 (规划中)
├── 分布式配置同步
├── 权限与审计系统
└── WebSocket 双向通信
```

## 如何贡献

1. 在修改架构前，先在此目录下创建 RFC 文档
2. RFC 需经过团队评审后方可实施
3. 实施完成后更新对应的架构文档
4. 归档旧版本文档，保持当前文档最新
